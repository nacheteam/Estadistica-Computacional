Ejercicio 2 (4.5 puntos)

Enunciado

Considera en el plano dos círculos centrados en el origen de radios 100 y 10 respectivamente. El primero representa una valla o cerca que impide el paso y el segundo un corral. Considera un animal que se encuentra en el punto de coordenadas (x1,y1) y se desplaza siguiendo las siguientes reglas: Si se encuentra fuera del círculo mayor, termina el proceso indicando "Se ha escapado". Si se encuentra dentro del círculo menor, termina el proceso indicando "Ha llegado al corral". En caso contrario se lanzan dos monedas. Si la primera moneda es cara, el animal intenta moverse una unidad a la derecha, si es cruz lo hace a la izquierda. Si la segunda moneda es cara, el animal intenta moverse una unidad hacia arriba, si es cruz lo hace hacia abajo. (Ambos movimientos se realizan simultáneamente, esto es, intenta desplazarse directamente a uno de los cuatro vértices del cuadrado que la rodea). Si como consecuencia de ese intento debe pasar del interior del círculo mayor a su exterior, no realiza ese movimiento. Escribe una función, f, que simule el movimiento del punto y devuelva: La progresión aleatoria definida, cuantos pasos han sido necesarios para terminar el proceso, que tipo de finalización se ha producido, realiza una simulación concreta del proceso y represéntala gráficamente. Escribe una función, Varios, con un argumento que permita repetir un número de veces, n, este proceso y devuelva un vector con el número de pasos necesarios para terminar el proceso en cada repetición. Utiliza esta función con el valor n=1000 y analiza los resultados

Solución

En primer lugar vamos a visualizar el recinto del que estamos hablando. 

dibuja_circunferencias <- function(R1=10, R2=100){
  f1 = function(x){sqrt(R1^2-x^2)}
  f2 = function(x){sqrt(R2^2-x^2)}
  plot(x=c(0,1), y=c(0,1), type="n", xlim=c(-R2,R2),ylim=c(-R2,R2))
  curve(f1(x),col="red", add=T)
  curve(-f1(x),add=T,col="red")
  curve(f2(x),col="red", add=T)
  curve(-f2(x),add=T,col="red")
}
dibuja_circunferencias()

Hagamos la función que describe el movimiento aleatorio. 

dentro_cfa <- function(x,y, r, Cx = 0, Cy = 0){
        return (((x-Cx)^2+(y-Cy)^2)<r^2)
}
posicion <- function(x1,x2,R1=10,R2=100){
  if(dentro_cfa(x1,x2,R1))
    return(1)
  else if(dentro_cfa(x1,x2,R2))
    return(2)
  else
    return(3)
}
movimiento <- function(x1=20,y1=20,n=1000){
  sucX <- 1:n
  sucY <- 1:n
  monedas1<-sample(c("cara","cruz"), n, replace=TRUE)
  monedas2<-sample(c("cara","cruz"), n, replace=TRUE)
  npasos<-n
  x<-x1
  y<-y1
  tipo_fin <- ""
    for(i in 1:n){
      sucX[i] = x
      sucY[i] = y
      if(posicion(x,y)==3){
        tipo_fin="Se ha escapado"
        npasos<-i
        break
      }
      else if(posicion(x,y)==1){
        tipo_fin="Ha llegado al corral"
        npasos<-i
        break
      }
      else{
        if(monedas1[i]=="cara")
          x=x+1
        else
          x=x-1
        if(monedas2[i]=="cara")
          y=y-1
        else
          y=y+1
      }
    }
  return(list(X=sucX, Y=sucY, Fin=tipo_fin, NumPasos=npasos))
}
mov<-movimiento()
